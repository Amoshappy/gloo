.. _glooctl:

glooctl
-------

CLI for Gloo

Synopsis
~~~~~~~~


glooctl is the unified CLI for Gloo.
	Find more information at https://solo.io

Options
~~~~~~~

::

  -h, --help          help for glooctl
  -i, --interactive   use interactive mode
